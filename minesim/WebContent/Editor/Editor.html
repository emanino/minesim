This quick video will give you an idea of how to "capture" the meaning of a sentence.
To complete this task follow those steps:
  1. Read carefully the sentence that is given to you. For example: "The menu of vegan restaurant A contains a tofu dish".
  2. Your goal is to recreate the meaning of this sentence as accurately as possible using the sentence snippets provided.
  3. Search suitable sentence snippets using the search option provided. For example, there might a snippet to describe a ""vegan restaurant".
  4. Once you have chosen a snippet, drag and drop it in your answer box.
  5. If the sentence given to you is an "if-then" sentence, use the "if" and "then" snippets to capture the condition and the consequence.
  6. These building blocks might require you to select variables, such as "A", "B" and "C".
  7. If the sentence you are given contains variables, you must use them in your snippets.
  8. If instead of a variable you want to specify a specific value (such as a number), choose the option "[...]" in the variable selection.
  9. Once you are satisfied that you captured the meaning of the sentence well, click on "Submit your answer".
  10. If you think it is not possible to capture the meaning of the sentence, click on "There is no good answer". You will be asked to specify which part of the sentence is problematic.

Important hints:
  1. When searching for snippets, try using synonyms. For example, you might not find a snippet that contains "more", but you can find one that contains "greater".
  2. If your search finds many results, you might have to scroll down to see them all.
  3. Don't worry if your solution contains some small grammatical mistakes, as long as the meaning is clear.
  4. If you make a mistake, you can always delete selected snippets by clicking on the bin icon on the right.
